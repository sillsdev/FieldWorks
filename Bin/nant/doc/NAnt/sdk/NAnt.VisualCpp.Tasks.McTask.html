Example

Compile text.mc using the default options. 


			<mc mcfile="text.mc"/>

Compile text.mc, passing a path to store the header, the rc file and some additonal options. 


			<mc mcfile="text.mc" headerpath=".\build" rcpath=".\build" options="-v -c -u"/>
