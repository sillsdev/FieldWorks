Remarks

If the file specified does not exist, the task will create it.

Example

Touch the Main.cs file. The current time is used.


			<touch file="Main.cs" />

Touch all executable files in the project base directory and its subdirectories. 


			<touch>
				<fileset>
					<include name="**/*.exe" />
					<include name="**/*.dll" />
				</fileset>
			</touch>
