# HermitCrab Synthesis by Gloss FLExTrans
Tools to run HermitCrab in synthesis mode using glosses for FLExTrans.

*GenerateHCConfig4FLExTrans* and *HCSynthByGloss* are 64-bit Windows programs.  They do not run on any other platform.

*GenerateHCConfig4FLExTrans* produces a Hermit Crab input file from a FLEx project that can be used to run Hermit Crab in synthesis mode.

*HCSynthByGloss* runs Hermit Crab in synthesis mode using an input file produced by *GenerateHCConfig4FLExTrans*.
